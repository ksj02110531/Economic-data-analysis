# 📊 경제지표 기반 경기 흐름 분석 미니 프로젝트

> 2019~2025년 한국 소비자물가지수 · 기준금리 · 원달러 환율 데이터를 Python으로 분석하고, Gemini AI로 경제 리포트를 생성하는 프로젝트

---

## 🎯 프로젝트 목적

전공 수업에서 경험한 Colab 데이터 분석 실습을 실제 경제지표 분석과 Gemini AI 리포트 생성 프로젝트로 확장하였다.

---

## 🛠️ 사용 기술

| 기술 | 용도 |
|------|------|
| Python | 전체 분석 로직 |
| pandas | 데이터 로드, 정제, 병합 |
| matplotlib | 시계열 그래프 시각화 |
| Google Colab | 클라우드 실행 환경 |
| Gemini AI API | 분석 결과 자동 요약 |

---

## 📁 데이터 출처

| 데이터 | 출처 | 기간 |
|--------|------|------|
| 소비자물가지수 (CPI) | 통계청 | 2019.01 ~ 2025.12 |
| 한국은행 기준금리 | 한국은행 ECOS | 2019.01 ~ 2025.12 |
| 원/달러 환율 | 한국은행 ECOS | 2019.01 ~ 2025.12 |

---

## 📂 파일 구조

```
📁 프로젝트 폴더
├── Economic_data_analysis.ipynb   # 메인 분석 노트북
├── 소비자물가지수.csv
├── 기준금리.csv
└── 환율.csv
```

## ▶️ 실행 방법

---

1. Google Colab에서 `Economic_data_analysis.ipynb` 열기
2. CSV 파일 3개 업로드 (`files.upload()` 셀 실행)
3. 셀을 순서대로 실행
4. Gemini API 키 입력 (`aistudio.google.com` 에서 API key 발급)
5. 이후 셀을 실행하면 AI 요약 리포트 생성
