# 📊 경제지표 기반 경기 흐름 분석 미니 프로젝트

> 2019~2025년 한국 소비자물가지수 · 기준금리 · 원달러 환율 데이터를 Python으로 분석하고, Gemini AI로 경제 리포트를 자동 생성하는 프로젝트입니다.

---

## 🎯 프로젝트 목적

경제학 전공자로서 **물가 · 금리 · 환율**이 개인과 기업의 의사결정에 미치는 영향에 주목했습니다.
단순히 데이터를 읽는 것에서 그치지 않고, Python으로 직접 정제·시각화하고, **AI 프롬프트 엔지니어링**을 통해 일반 사용자도 이해하기 쉬운 경제 리포트를 생성하는 흐름을 구현했습니다.

이 프로젝트는 **SKALA 서류 면접 기술 활용 능력** 증빙을 위해 제작되었습니다.

## 🔄 분석 흐름

```
1. CSV 데이터 로드
       ↓
2. 컬럼명 통일 및 날짜 타입 변환
       ↓
3. 세 데이터셋 date 기준 병합 (inner join)
       ↓
4. 전년동월 대비 CPI 상승률 계산 (YoY)
       ↓
5. 시계열 그래프 시각화 (CPI / 금리 / 환율)
       ↓
6. 핵심 통계값 추출 (시작·끝·최소·최대)
       ↓
7. Gemini AI에 프롬프트 전달 → 경제 리포트 생성
```

## 📈 주요 분석 내용

### 데이터 전처리 핵심 포인트

- **컬럼명 통일**: 세 CSV의 컬럼명이 달라 `rename()`으로 `date`, `cpi`, `rate`, `usd_krw`로 통일
- **날짜 타입 변환**: 문자열로 저장된 날짜를 `pd.to_datetime()`으로 변환
- **환율 숫자 변환**: 쉼표(`,`)가 포함된 환율 데이터를 `str.replace()`로 제거 후 `float` 변환
- **YoY 계산**: `shift(12)`로 12개월 전 값을 참조해 전년동월 대비 상승률 산출

### 시각화
CPI 추이, 기준금리 추이, 원/달러 환율 추이를 각각 시계열 그래프로 표현하고, 한글 폰트(NanumGothic) 설치로 깨짐 문제를 해결했습니다.

## 🤖 Gemini AI 프롬프트 설계

분석 수치를 바탕으로 아래 조건을 반영한 프롬프트를 구성했습니다.

1. 물가·금리·환율 흐름을 **쉬운 언어**로 설명
2. **일반 소비자 및 취업준비생** 관점의 시사점 포함
3. 데이터만으로 확인 불가한 인과관계는 **단정하지 않음**
4. "영향을 줄 수 있다", "가능성이 있다" 등 **신중한 표현** 사용

## 🚧 문제 해결 기록

| 문제 | 원인 | 해결 방법 |
|------|------|-----------|
| 그래프 한글 깨짐 | Colab 기본 폰트가 한글 미지원 | `fonts-nanum` 설치 후 `rcParams`로 폰트 변경 |
